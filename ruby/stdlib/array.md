# Arrayクラス

## product

レシーバの配列と引数で与えられた配列から要素を1個ずつとって配列にして返す

```ruby
[1, 2].product([3, 4])
# => [[1, 3], [1, 4], [2, 3], [2, 4]]
```

## each_cons(n)

要素を重複ありでn要素ずつに区切り、ブロックに渡して繰り返す

```ruby
(1..5).each_cons(3){|v| p v }
#=> [1, 2, 3]
# [2, 3, 4]                                                                
# [3, 4, 5]            
```

## each_slice(n)

each_consと似ているが、each_sliceは要素を重複させない

```ruby
(1..5).each_slice(3){|v| p v }
# => [1, 2, 3]
# [4, 5]
```
