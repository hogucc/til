# Dateクラス

## 日付のフォーマット

- %x: 日付 (%m/%d/%y)
  - 2015年1月15日なら、 `01/05/15`
- %X: 24時間制の時刻（%H:%M:%S）
  - 10時9分23秒なら `10:09:23`
- %m: 月（01-12）
- %M: 分（00-59）
- %d: 日（01-31）
- %D: `%x` と同じ
- %y: 西暦の下2桁（00-99）
- %Y: 西暦を表す数
- %h: 月の省略名（Jan,Feb...）
- %H: 24時間制の時（00-23）
- %s: 1970-01-01 00:00:00 UTC からの経過秒
- %S: 秒(00-60) (60はうるう秒)

年は西暦で表記したい場合が多い気がするので、`%y`より`%Y`を使う機会が多そう

## to_s

`2022-02-07`形式で日付を出力する
